How to Play:

  • Click blocks: Click on groups of 2 or more same-colored blocks to clear them
  • Goal: Clear as many blocks as possible to score points
  • Scoring: Larger groups give more points (2 blocks = 10pts, 3 blocks = 30pts, 4+ blocks = 50pts each)
  • Level Up: Clear a certain number of blocks to advance to the next level
  • Game Over: The game ends when no more moves are available
  • P: Pause/Resume game
